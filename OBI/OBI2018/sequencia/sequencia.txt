Sequência

Nome do arquivo: “sequencia. x”, onde x deve ser clcpplpasljavaljslpy2lpy3

O professor da importante disciplina de Indução Matemática está tentando resolver uma versão
generalizada de um problema muito tradicional: encontrar o valor máximo possível para a soma dos
elementos de uma subsequência contígua de uma sequência de números inteiros quaisquer. Mais
rigorosamente, dado uma sequência S = [s1,82,...,sn], onde s; é um número inteiro qualquer, para
1<i<N, maximizar soma(i, j) = s; + siyi +--- + s; entre todos os possíveis pares (i, j), onde
I<i<j<N.

Na versão do professor, entretanto, alguns elementos da sequência são especiais e estão marcados.
Além da sequência marcada, são dadas como entrada duas cotas: Le H, com L<H. O objetivo
agora é encontrar o valor máximo possível para a soma dos elementos de uma subsequência contígua,
que contenha pelo menos L e no máximo H elementos marcados.

Por definição, uma subsequência vazia (de zero elementos) tem soma igual a zero. Mas note que,
como podemos ter uma cota inferior para o número de elementos marcados, a subsequência contígua
de soma máxima pode ter soma negativa!

Entrada

A primeira linha da entrada contém três inteiros N, Le H, indicando respectivamente o número de
elementos na sequência, a cota inferior L e a cota superior H. A segunda linha contém N inteiros
s; para l <i< N, definindo os elementos da sequência. A terceira linha contém N inteiros mi,
paral <i<N, indicando as marcas. Se o i-ésimo elemento está marcado, o valor é m; = 1. Se
não estiver marcado, m; = 0.

Saída

Imprima um inteiro, representando o valor máximo possível para a soma dos elementos de uma
subsequência contígua, que contenha pelo menos L e no máximo H elementos marcados.

Restrições
c1<N<IG
eO<L<H<9
e IO <s;<10,paral<i<N
e O número de elementos marcados na sequência é maior ou igual a L; portanto sempre existe

solução.
